Apartment Prices Analysis in Poland 
📌 Про проєкт

Аналітичний проєкт з дослідження ринку нерухомості Польщі на основі даних про продаж квартир у найбільших містах країни.

Мета проєкту:

провести очищення та підготовку даних;
виконати дослідницький аналіз (EDA);
виявити фактори, що впливають на вартість квартир;
створити інтерактивний дашборд для візуалізації результатів.
🛠️ Використані інструменти
SQL (Google BigQuery)
Python (Pandas, NumPy, Matplotlib, Seaborn)
Power BI
GitHub

🔍 Етапи роботи
1. Очищення даних
перевірка пропущених значень;
пошук дублікатів;
перевірка типів даних;
очищення та стандартизація полів.
2. SQL-аналіз

Проведено аналіз:

середньої ціни квартир;
середньої ціни за м²;
розподілу квартир за містами;
впливу кількості кімнат;
впливу паркінгу та ліфта на вартість.
3. Python EDA

Виконано:

статистичний аналіз;
побудову гістограм;
аналіз кореляцій;
дослідження розподілів даних.
4. Power BI Dashboard

Створено інтерактивний дашборд з:

KPI-картками;
аналізом цін по містах;
розподілом квартир за кількістю кімнат;
впливом характеристик житла на ціну.
📊 Основні висновки
Варшава має найвищу середню ціну за м².
Краків та Гданськ займають друге та третє місця за вартістю житла.
Квартири з ліфтом мають вищу середню ціну за м².
Наявність паркінгу збільшує середню вартість квартири приблизно на 114 тис. PLN.
Найбільшу частку ринку становлять двокімнатні та трикімнатні квартири.
